# Calcifer
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'calcifer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install calcifer

## Usage

`bundle exec calcifer | xargs bundle exec rspec`
